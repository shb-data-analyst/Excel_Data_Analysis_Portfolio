# Excel_Data_Analysis_Portfolio

Excel dashboards and data analysis portfolio using pivot tables, charts, and Power Query.

This repository contains Excel-based data analysis projects focused on dashboards, automation, and business insights.

## Projects

### 1. Sales Performance Analysis
- Sales trend and regional analysis  
- Interactive Excel dashboard using pivot tables and charts  
- KPI-based performance tracking  

### 2. Expense & Budget Analysis
- Budget vs actual expense tracking  
- Variance analysis with conditional formatting  
- Interactive Excel dashboard with KPIs and slicers  

### 3. Retail Store Sales Analysis
- End-to-end Excel project combining data cleaning, advanced formulas, and dashboards  
- Interactive sales dashboard with KPIs, slicers, and business insights  

### 4. Customer Order Analysis using Power Query & Excel Dashboard
- Automated data cleaning and transformation using Power Query  
- Merging Orders and Customers datasets using Left Join and Inner Join  
- Handling missing customer records by replacing null values  
- Refresh-ready data model with one-click update  
- Interactive Excel dashboard with KPIs, charts, slicers, and conditional formatting
