# Retail Store Sales Analysis Dashboard (Microsoft Excel)

## Project Overview
This project analyzes retail store sales data to understand regional performance, product category contribution, customer behavior, and sales trends. The goal is to convert raw sales data into meaningful business insights using Microsoft Excel.

## Dataset
- Simulated retail sales dataset
- Customer master data for lookup and classification
- Includes order status, region, category, and sales amount

## Tools Used
- Microsoft Excel

## Skills & Techniques Applied
- Data Cleaning (TRIM, CLEAN)
- XLOOKUP for data merging
- IF, IFERROR for business logic
- SUMIFS & COUNTIFS for conditional analysis
- Pivot Tables & Pivot Charts
- KPI creation
- Interactive dashboard with slicers
- Trend and category analysis

## Dashboard Features
- Total Sales, Completed Sales, and Completed Orders KPIs
- Sales by Region (Column Chart)
- Sales Distribution by Category (Donut Chart)
- Monthly Sales Trend (Line Chart)
- Interactive slicers for Region, Category, and Order Status

## Files Included
- `Retail_Store_Analysis.xlsx`
- Dashboard and analysis screenshots

## Key Insights
- West region generates the highest completed sales.
- Electronics category contributes the majority of revenue.
- Returning customers drive a significant portion of sales.
- Pending and cancelled orders reduce realized revenue.
- Sales trends indicate seasonal fluctuations.
